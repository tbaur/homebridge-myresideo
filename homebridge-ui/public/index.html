Link your Resideo account

Your Resideo account is linked. A refresh token is saved in your configuration and the plugin keeps it refreshed automatically. Re-link only if you rotate your developer credentials or want to connect a different account.

This connects the plugin to your Resideo / Honeywell Home account using OAuth2. You will need a Resideo developer application (created at developer.honeywellhome.com) for the Consumer Key and Secret below. The plugin only ever stores OAuth tokens, never your account password.

Must match the Callback URL registered on your Resideo developer application, byte-for-byte. This default is fine for most setups; you do not need anything running at this address.

Opens Resideo's sign-in page in a new tab. After you approve access, your browser is redirected to the Redirect URL above with a one-time code attached. The page may show an error or a blank page — that is expected; nothing needs to be listening there.
